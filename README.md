# IDX Web

A browser-only version of [IDX](https://idx.xyz/docs/libs-idx) using [3ID Connect](https://github.com/3box/3id-connect).

## Installation

```sh
npm install @ceramicstudio/idx-web
```

## Documentation

- [IDX documentation](https://idx.xyz)
